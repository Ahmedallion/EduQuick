# Get Help

If you are experiencing issues with EduQuick or have suggestions for new features, please use the resources below. Providing clear details helps us resolve problems faster.

---

## Reporting Issues

The primary way to report bugs or technical issues is through our GitHub repository. This allows us to track progress and keep you updated on fixes.

[Report an Issue on GitHub](https://github.com/Ahmedallion/EduQuick/issues){: .btn .btn-purple target="\_blank" rel="noopener"}

### What to include in your report:

To help us diagnose the problem, please include the following information in your GitHub issue:

- **Browser:** Name and version (e.g., Chrome v121).
- **Context:** The specific page or quiz type where the error occurred.
- **Description:** What you expected to happen vs. what actually happened.
- **Error Codes:** Any specific codes shown (e.g., `EQ-001`).
- **Visuals:** A screenshot of the error or the console log if possible.
