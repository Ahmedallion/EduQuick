## Reporting Issues

Include:

- Browser name & version
- Page / quiz type
- Expected outcome vs actual outcome
- Error code if shown (e.g., `EQ-001`)
- Screenshot if possible

{: .note }

> Reporting issues with complete details helps improve EduQuick faster!
