# Quick Start

Follow these steps to get EduQuick running quickly.

---

## Step 1: Download Bookmarklet

[Download EduQuick HTML](https://github.com/Ahmedallion/EduQuick/install/eduquick.html){: .btn .btn-purple target="\_blank" rel="noopener"}

---

## Step 2: Open Bookmarks Manager

- Chrome / Edge: Type `chrome://bookmarks` or `edge://favorites`

---

## Step 3: Import the Bookmark

- Click the three dots → **Import bookmarks**
- Select the downloaded `eduquick.html`

---

## Step 4: Run EduQuick

1. Go to [Educake](https://www.educake.co.uk){:target="\_blank" rel="noopener"}
2. Log in
3. Click **EduQuick** from your bookmarks bar

---

{: .note }

> Tip: If this doesn’t work, check [Installation](/installation){: .btn .btn-purple} for your browser/device.
