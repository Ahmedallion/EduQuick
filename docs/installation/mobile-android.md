# Installing on Android (Limited Support)

Android browsers do not properly support bookmarklets.

{: .warning }

> Warning: This method is **strongly discouraged** and may not work.

---

## Steps

1. Bookmark the EduQuick repository
   ![Android Step 1](../assets/images/installation/android-step1.png)

2. Each time you want to run EduQuick:
    - Copy JS from [`eduquick.js`](https://github.com/Ahmedallion/EduQuick/install/eduquick.js){: .btn .btn-purple target="\_blank" rel="noopener"}
      ![Android Step 2-1](../assets/images/installation/step-copy.png)
    - Log into Educake
    - Clear the address bar, paste JS (starts with `javascript:`), press Go
      ![Android Step 2-3](../assets/images/installation/android-step2-3.png)

---

{: .note }

> Tip: Android bookmarklets must be repeated each time and may break after browser updates.
