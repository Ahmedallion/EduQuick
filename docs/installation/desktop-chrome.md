# Installing on Chrome & Chromium Browsers

Covers Chrome, Brave, Opera, Vivaldi.

---

## Steps

1. [Download EduQuick HTML](https://github.com/Ahmedallion/EduQuick/install/eduquick.html){: .btn .btn-purple target="\_blank" rel="noopener"}
   ![Chrome Step 1](../assets/images/installation/step-download.png)

2. Open bookmarks manager: `chrome://bookmarks`

3. Enable bookmarks bar: `Ctrl + Shift + B`

4. Click three dots → **Import bookmarks**
   ![Chrome Step 4-1](../assets/images/installation/chrome-step4-1.png)
   ![Chrome Step 4-2](../assets/images/installation/chrome-step4-2.png)

5. Select HTML file: `eduquick.html`
   ![Chrome Step 5](../assets/images/installation/chrome-step5.png)

6. EduQuick appears in bookmarks bar

---

{: .note }

> Tip: Other Chromium browsers follow these same steps.
> Make sure you are logged into **Educake** before running EduQuick.
