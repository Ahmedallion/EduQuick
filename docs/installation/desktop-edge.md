# Installing on Microsoft Edge

---

## Steps

1. [Download EduQuick HTML](https://github.com/Ahmedallion/EduQuick/install/eduquick.html){: .btn .btn-purple target="\_blank" rel="noopener"}
   ![Edge Step 1](../assets/images/installation/step-download.png)

2. Open favorites manager: `edge://favorites`

3. Enable favorites bar → Always
   ![Edge Step 3](../assets/images/installation/edge-step3.png)

4. Click three dots → **Import favorites** → **Import**
   ![Edge Step 4-1](../assets/images/installation/edge-step4-1.png)
   ![Edge Step 4-2](../assets/images/installation/edge-step4-2.png)

5. Select HTML file: `eduquick.html`
   ![Edge Step 5](../assets/images/installation/edge-step5.png)

6. EduQuick appears in favorites bar

---

{: .note }

> Tip: Always log in to **Educake** before running EduQuick.
