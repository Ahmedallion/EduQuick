# Installing on iOS (Safari)

---

## Steps

1. Open [`eduquick.js`](https://github.com/Ahmedallion/EduQuick/install/eduquick.js){: .btn .btn-purple target="\_blank" rel="noopener"}

2. Copy **all** the code
   ![iOS Step 2](../assets/images/installation/step-copy.png)

3. Bookmark the page
   ![iOS Step 3-1](../assets/images/installation/ios-step3-1.png)
   ![iOS Step 3-2](../assets/images/installation/ios-step3-2.png)

4. Open bookmarks → Edit the new bookmark
   ![iOS Step 4](../assets/images/installation/ios-step4.png)

5. Rename it **EduQuick**, replace URL with copied JS → Save
   ![iOS Step 5](../assets/images/installation/ios-step5.png)

---

## Running EduQuick

- Log in to Educake
- Rotate device **horizontally**
- Zoom out fully
- Open bookmarks → tap **EduQuick**
