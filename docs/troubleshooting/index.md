# Troubleshooting

If something is not working, check the guides below to resolve common issues.

---

### Need help with setup?

If the tool isn't appearing in your bookmarks bar or the script fails to launch after clicking, start with our dedicated installation troubleshooting guide.

[Installation Troubleshooting](/installation){: .btn .btn-purple }

<br>

{: .note }

> **Common Fix:** Many issues are caused by the browser automatically removing the `javascript:` prefix when pasting a bookmark. Always double-check your bookmark URL if the script does not execute.
